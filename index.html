About Bio-Scan Image Recognition

The Bio-Scan module utilizes a convolutional neural network (MobileNet V1) trained on millions of images. The model operates 100% locally inside your web browser using WebGL GPU acceleration via TensorFlow.js. When you load a food specimen photo, the network extracts spatial feature maps and returns the top 3 corresponding ImageNet labels, which are mapped to our organic library for complete chemical breakdowns.
